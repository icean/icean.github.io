XX卡破解 - 相关字段查找

发表于 2017-02-09 | 更新于 2019-09-22 | 分类于 其他 |
使用ACR-122对XX卡进行破解。2008 年的时候，荷兰恩智浦（NXP）公司开发的 RFID 产品 Mifare Classic 就被破解了，详细请自行Google 或百度。RFID即为射频识别，NFC近场通信，实际上NFC可以理解为以RFID技术为基础的一种产品。

2. 相关字段查找

已知信息

  • 已知卡片导出的dump文件前四个字节为卡片的出场UID，第五个字节为校验位
  • 经过测试，0x91、0x92数据存储位，0x93对0x91、0x92的一个校验位

测试思路

  • 找到两张可以正常使用的卡片，分别导出dump文件
  • 将导出的dump文件的0x01~0x04（UID和校验位）修改为相同数值，将0x91~0x93（数据存储位和校验位）
  • 对比两个文件的不同

测试结果

  • 为了简化操作，将两个dump文件的0x00~0x04与0x91~0x93全部置为0

    由此可以发现

    • 同一张卡片内的0x82、0x83与0xD9、0xDA相同
    • 不同卡片的0x82、0x83与0xD9、0xDA不同
    • 不同卡片的0x9B~0x9D不同
    • 卡片的其他字段，与UID等信息无关

猜测

现在猜测有几种可能

  • 这几个字段与卡号（卡片表面的数字）有关
  • 这几个字段与卡片UID有关系
  • 未知控制信息

修改测试
