OpenWRT Python安装 & 网络验证自动登录

发表于 2017-03-31 | 更新于 2018-05-08 | 分类于 技术 |

1. 需求分析

实验室对外部网络的访问，需要登录网页进行验证。但是在每天凌晨3点左右会自动登出。因此每天早上，都需要重新进行网页验证，虽然这不耗费什么时间精力，但还是觉得有瑕疵，因此决定在OpenWRT上实现自动登录外部网络访问验证。

最简单粗暴的办法，还是用Python！先谢这位同学的脚本，https://github.com/KDF5000/ICTAuth

但是因为硬件存储空间不够大，所以不能直接把Python安装在路由器本机的存储空间！所以要另辟蹊径！

添加方式很简单，直接到其他装有Python的Linux主机/usr/lib/python2.7/site-packages目录下，拷贝requests库相关的文件，然后粘贴到/mnt/sda1packages/usr/lib/python2.7/site-packages目录。
